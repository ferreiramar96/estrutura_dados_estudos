# Estudos de Estrutura de Dados e Algoritmos

Este repositório contém meus estudos e implementações de diversas estruturas de dados e algoritmos fundamentais, utilizando **Python** e **Jupyter Notebooks**. O objetivo é servir como uma base de consulta e prática para os conceitos essenciais da Ciência da Computação.

---

## 📂 Organização do Repositório

O conteúdo está organizado por categorias, facilitando a navegação entre os diferentes temas:

### 1. 📊 Teoria e Análise
- **[Notação Big O](Notação_Big_O.ipynb)**: Introdução à análise de complexidade de tempo e espaço.

### 2. 🧱 Estruturas de Dados Lineares
- **[Vetores](Vetores%20(Ordenados%20e%20Não))**: Implementações de vetores ordenados e não ordenados.
- **[Listas Encadeadas](Listas%20Encadeadas)**:
    - Lista encadeada simples.
    - Lista encadeada simples com extremidade dupla.
    - Lista duplamente encadeada.
- **[Pilhas, Filas e Deques](Pilhas,%20Filas%20e%20Deques)**: Implementações clássicas dessas estruturas.

### 3. 🔄 Algoritmos e Lógica
- **[Recursão](Recursao)**: Conceitos e exemplos práticos de funções recursivas.
- **[Algoritmos de Ordenação](Algoritmos%20Ordenacao)**: Bubble Sort, Selection Sort, Insertion Sort, etc.

### 4. 🌳 Estruturas de Dados Não Lineares
- **[Árvores](Arvores)**: Árvores Binárias de Busca (BST).
- **[Grafos](Grafos)**:
    - Implementação básica de Grafos.
    - Algoritmo de Dijkstra (Menor Caminho).
    - Busca Gulosa.
    - Busca A* (A-Estrela).
